# CCSKEL (C/C++ Skeleton)

CCSKEL — это легковесная консольная утилита для автоматической генерации готовой структуры шаблонов проектов на языках C и C++. Инструмент избавляет от рутинной настройки каталогов, конфигурационных файлов сборки и базового исходного кода.

## 🚀 Основные возможности

* **Два стандарта:** Полная поддержка генерации структуры как для чистого C, так и для C++.
* **Готовая сборка:** Автоматическое создание конфигурационных файлов CMakeLists.txt или Makefile.
* **Изоляция кода:** Разделение проекта на логические каталоги (`src`, `include`, `external`, `build`).
* **Быстрый старт:** Создание базового файла `main.c` / `main.cpp` для мгновенной проверки работоспособности.

## 📁 Генерируемая структура проекта

При запуске утилита создает следующую иерархию директорий и файлов:

```text
my_project/
├── Makefile (или CMakeLists.txt)
├── assets
├── build
│   ├── bin
│   │   ├── assets
│   │   │   ├── 1.jpeg
│   │   │   └── arial.ttf
│   │   ├── lib
│   │   │   ├── libraylib.a
│   │   └── program
│   └── obj
│       └── main.o
├── external
│   ├── include
│   │   ├── raylib.h
│   │   ├── raymath.h
│   │   └── rlgl.h
│   └── lib
│       └── libraylib.a
├── include
│   └── main.h (или .hpp)
└── src
    └── main.c (или .cpp)
```

## 🛠 Требования к среде

Для сборки и запуска самого генератора CCSKEL вам понадобятся:
* Компилятор GCC/CLANG
* Система сборки Make

## 📦 Сборка и установка

1. Склонируйте репозиторий с исходным кодом:
   ```bash
   git clone https://github.com/ichepyxa/ccskel
   cd ccskel
   ```

2. Скомпилируйте проект:
   ```bash
   make
   ```

## 💻 Использование

Запустите утилиту, передав директорию и необходимые флаги конфигурации:

```bash
ccskel [опции] <директория>
```

### Доступные флаги и параметры

* `--cpp` — Сгенерировать структуру шаблона для проекта на C++.
* `--сmake` — Использовать CMake вместо Makefile в качестве системы сборки.
* `-h`, `--help` — Показать справку по использованию программы.

### Примеры команд

Создание C++ проекта в текущей директории с использование Makefile:
```bash
ccskel --cpp .
```

Создание C проекта в текущей директории с использованием CMake:
```bash
ccskel --cmake .
```
